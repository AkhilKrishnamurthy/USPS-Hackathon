Mail Trigger Application:

Back end:
Open the backend with command prompt, and type "node index.js"

Front end:
open the frond end with command prompt, and type "npm start"

Enter your email id and any characters as password you prefer, and it would trigger a mail to your email about invalid credentials.

Integrate the backend with your application for strenthening the security of the account.


Technology Stacks used:
Front end: React.JS
Backend: Node.JS
Packages: nodemailer
